### Project Purpose
    The purpose of this project is to design and develop a professional, version controlled personal introduction webpage utilizing basic Markdown, HTML and CSS languages. 

### Project Description
    This project will demostrate the knowledge gained during the first four weeks of class.
        1. Title element in the head.
        2. A main heading (h1) with name.
        3. A subheading (h2) for my major and/or professional interests
        4. At least two paragraphs (p) describing professional goals for the coarse and career
        5. An unordered list (ul) with at least three list items (li) detailing hobbies and interests
        6. A hyperlink (a) to my GitHub profile.
    The final project will be able to be viewed at https://kedaily.github.io     

### Desired Results
    The project should be able to convey the following results
*   A fully functional personal introduction page written in HTML.
*   A clear understanding  of basic HTML syntax and structure
*   A document that can be shared online via GitHub.
