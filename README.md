## Project Purpose
    The purpose of this project is to design and develop a perfessional, version controlled website for Daily's Wreck 'Em Rabbitry.

### Project Description
    This project will demostrate the knowledge gained during the entire semester.
        1.Title element in the head.
        2. A main heading (h1) with Daily's Wreck 'Em Rabbitry.
        3. Daily's Wreck 'Em Rabbitry Logo in the Header, centered
        4. https://kedaily.github.io will show Index.html with link at the bottom to about.html page. 
        5. Pictures, DOBs descriptions and any show records to be on index.html. 
        6. Will use the Texas Tech University color scheme of Black #000000, Scarlet #E90802, and White #FFFFFF. Complementary colors used are Light Grey #EDEDED, Dark Grey #333333, and Medium Grey #757575
        7. Pictures will implement padding to seperate between each rabbit to allow easier flow of information. 
        8. Using <div class="container"> and <div class="text-content"> to have the rabbits information beside their pictures. 
        9. Links between index.html and about.html is in the footer and will be labled "Our Rabbits" and "About Us".
        10. Cassidy's favorite scripture to be in the footer to give glory to God for all he allows us to do thru him. 
        11. About Us page to have a basic introduction to Daily's Wreck 'Em Rabbitry. No contact information will be available on this version of the website.
